# Button

Common button component.

## Usage

```vue
<template>
  <MaButton>Click me</MaButton>
  <MaButton variant="outline">Outline</MaButton>
</template>
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| variant | `string` | `solid` | Button style variant |
| color | `string` | `primary` | Button color theme |
| size | `string` | `medium` | Button size |
